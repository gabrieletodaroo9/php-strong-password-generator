# Strong Password Generator 

Un'applicazione web realizzata in **PHP** che genera password sicure e personalizzabili in base alle preferenze dell'utente (lunghezza e tipologia di caratteri).

Il progetto mette in pratica la logica di generazione algoritmica, l'uso delle **Sessioni PHP** per la persistenza dei dati e il reindirizzamento delle pagine (Redirect).
